# Registration Landing Page with Dynamic Timer

Профессиональный лендинг с адаптивной формой регистрации, динамическим счетчиком обратного отсчета и кастомным графическим декором на чистом CSS/JS.

## 🚀 Функционал

- **Dynamic Countdown:** Таймер обратного отсчета, создающий эффект срочности.
- **Dual Form System:** Полноразмерная форма для десктопа и компактная модальная форма для мобильных устройств.
- **Custom UI Components:** Уникальные чекбоксы, стилизованные теги с декоративными углами и кастомная валидация полей.
- **Advanced CSS Layout:** Использование CSS Grid для построения сложной фоновой сетки и абсолютного позиционирования элементов декора (формулы, фигуры).
- **Fully Responsive:** Адаптивная верстка (Desktop -> Tablet -> Mobile).

## 🛠 Технологии

- **HTML5** — семантическая структура.
- **CSS3** — Grid, Flexbox, кастомные свойства (переменные), сложные анимации.
- **JavaScript (Vanilla)** — логика таймера, управление модальными окнами и валидация форм без сторонних библиотек.

## 📁 Структура проекта

- `index.html` — основная разметка страницы.
- `style.css` — все стили, разделенные на логические секции (Hero, Form, Modal, Media).
- `script.js` — логика работы таймера и обработка событий (открытие/закрытие модалки).
- `/images` — графические ассеты (логотип, SVG-декор, формулы).

## ⚡️ Быстрый старт

1. Склонируйте репозиторий или скачайте файлы.
2. Откройте `index.html` в любом современном браузере.
3. Для редактирования стилей используйте файл `style.css` (настройки цветов и шрифтов вынесены в `:root`).

## 🎨 Особенности верстки

- **Сетка:** Фон реализован через `linear-gradient`, что позволяет сетке бесконечно масштабироваться без использования тяжелых картинок.
- **Модальное окно:** Реализовано с эффектом `backdrop-filter: blur`, обеспечивающим современный вид.
- **Валидация:** Поля ввода подсвечиваются красным при ошибке, обеспечивая качественный UX.
