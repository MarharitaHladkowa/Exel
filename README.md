# Registration Landing Page with Dynamic Timer

Професійний лендинг з адаптивною формою реєстрації, динамічним лічильником зворотного відліку та кастомним графічним декором на чистому **CSS/JS**.

---

## 🚀 Функціонал

- **Dynamic Countdown:** Таймер зворотного відліку, що створює ефект терміновості.
- **Dual Form System:** Повнорозмірна форма для десктопа та компактна модальна форма для мобільних пристроїв.
- **Custom UI Components:** Унікальні чекбокси, стилізовані теги з декоративними кутами та кастомна валідація полів.
- **Advanced CSS Layout:** Використання _CSS Grid_ для побудови складної фонової сітки та абсолютного позиціонування елементів декору (формули, фігури).
- **Fully Responsive:** Адаптивна верстка (Desktop -> Tablet -> Mobile).

---

## 🛠 Технології

- **HTML5** — семантична структура.
- **CSS3** — Grid, Flexbox, кастомні властивості (змінні), складні анімації.
- **JavaScript (Vanilla)** — логіка таймера, керування модальними вікнами та валідація форм без сторонніх бібліотек.

---

## 📁 Структура проєкту

- `index.html` — основна розмітка сторінки.
- `style.css` — усі стилі, розділені на логічні секції (Hero, Form, Modal, Media).
- `script.js` — логіка роботи таймера та обробка подій (відкриття/закриття модалки).
- `/images` — графічні активи (логотип, SVG-декор, формули).

---

## ⚡️ Швидкий старт

1.  **Клонування:** Склонуйте репозиторій або завантажте архів з файлами.
2.  **Запуск:** Відкрийте `index.html` у будь-якому сучасному браузері.
3.  **Налаштування:** Для редагування стилів використовуйте файл `style.css` (основні кольори та шрифти винесені у `:root`).

---

## 🎨 Особливості верстки

- **Сітка:** Фон реалізований через `linear-gradient`, що дозволяє сітці нескінченно масштабуватися без використання важких зображень.
- **Модальне вікно:** Реалізовано з ефектом `backdrop-filter: blur`, що забезпечує сучасний "скляний" вигляд.
- **Валідація:** Поля введення динамічно підсвічуються червоним у разі помилки, забезпечуючи якісний UX.
